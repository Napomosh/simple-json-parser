◆ separate_next_element()

void parser::separate_next_element ( std::string &  buffer)
private

This private method separate next elements. Also it controls iterator (increment it when it is necessary). Only two options are possible: a separator and a word. But any words starts with quotes (separators). This function determines which separator is marked with iterator (cur_position) using method separator_type(). This method will determine what to do depending on the type of separator. These actions are described in other private methods do_action_xxx()

Parameters
buffer- in this string we will save the extracting word or separator
